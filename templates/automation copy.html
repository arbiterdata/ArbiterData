Problems with Excel

Excel is one of the greatest software packages ever built. Its flexibility, power and ease of adoption has made it a cornerstone business processes. It can be used to store data, process calculuations and generate reporting. However it suffers from a variety of weaknesses which make it an unreliable, slow and risky tool.

Data Integrity

due to how Excel is designed it is impossible to be confident that data in the spreadsheet is robust. Anybody who has accidentally deleted or modified data

Speed

as spreadsheets grow and have functionality added to them they slow down and eventually become a drain on productivity as users have to wait for Excel to process even very small changes

Manual

to make Excel do anything users have to open it and import & cleanse data, modify formulas and format the output. These manual processes are error prone, time consuming

Traceability

due to the flexibility of Excel inputs, processing and outputs can be difficult (and in some circumstances) impossible to trace. This lack of tracability means spreadsheets can become difficult or impossible to be managed by anybody other than who created it in the first instance

Python Excel automation

Python is a programming language with Excel like modules. It has all of the data processing capability of Excel and has all the formatting capabilities of Excel. Due to Python's power processses that can take hours in Excel can be completed in seconds in Python.

Python in the cloud

Most organisations do not have suitable IT infrastructure to host Python programs. At Arbiter Data we resolve this problem by building cloud servers to host Python programs which means busineses can access their Python programs via a website.

Hallmarks of potential Python migration

Oonce a spreadsheet is used repetitively, automation should be considered. Spreadsheets with the following hallmarks are good candidates for automation

  • Slow to open
  • Unresponsive to changes in data & formulas
  • Various inputs
  • Vlookups
  • Index Matching
  • Sum Ifs
  • Pivot Tables
  • Multiple tabs
  • Graphical outputs
  • Summary table outputs
  • Calculators
